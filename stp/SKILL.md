## 1. Segmentation (시장세분화)

Divide the total market into distinct subgroups that share similar needs, characteristics, or behaviors.

### 1.1 Segmentation Bases

#### Geographic (지리적 세분화)
| Variable | Examples |
|----------|----------|
| Region (지역) | 수도권, 영남권, 호남권, 충청권 |
| City size (도시규모) | 대도시, 중소도시, 농어촌 |
| Climate (기후) | 온대, 아열대, 산간 |
| Density (인구밀도) | 도심, 교외, 산업단지 |

#### Demographic (인구통계적 세분화)
| Variable | Examples |
|----------|----------|
| Age (연령) | Z세대, 밀레니얼, X세대, 베이비부머 |
| Income (소득) | 저소득, 중산층, 고소득, 초고소득 |
| Occupation (직업) | 전문직, 사무직, 자영업, 생산직 |
| Education (학력) | 고졸, 대졸, 대학원 이상 |
| Family lifecycle (가족생애주기) | 미혼, 신혼, 자녀양육기, 빈둥지 |

#### Psychographic (심리적 세분화)
| Variable | Examples |
|----------|----------|
| Lifestyle (라이프스타일) | 건강지향, 가치소비, 편의추구, 경험중시 |
| Personality (성격) | 혁신적, 보수적, 사교적, 독립적 |
| Values (가치관) | 환경의식, 가성비중시, 프리미엄지향 |
| Social class (사회계층) | 상류, 중상, 중류, 중하, 하류 |

#### Behavioral (행동적 세분화)
| Variable | Examples |
|----------|----------|
| Usage rate (사용량) | 비사용자, 경사용자, 중사용자, 다사용자 |
| Loyalty (충성도) | 절대충성, 이중충성, 전환형, 무충성 |
| Benefits sought (추구편익) | 품질, 가격, 편의성, 서비스, 지위 |
| Purchase occasion (구매상황) | 정기구매, 특별행사, 긴급구매 |
| Readiness stage (구매준비단계) | 인지, 관심, 평가, 시용, 채택 |

#### B2B Segmentation (B2B 세분화)

**Firmographic (기업특성)**
| Variable | Examples |
|----------|----------|
| Industry (산업) | 석유화학, 반도체, 제지, 발전, 제철 |
| Company size (기업규모) | 대기업, 중견기업, 중소기업 |
| Revenue (매출규모) | 100억 미만, 100~1000억, 1000억 이상 |
| Location (위치) | 울산, 여수, 대산, 이천, 청주 |
| Procurement approach (구매방식) | 장기계약, 단가계약, 건별발주, 입찰 |

**Needs-based (니즈 기반)**
| Needs Cluster | Description |
|---------------|-------------|
| Price-sensitive (가격민감형) | 최저가 입찰, 비용절감 우선 |
| Quality-first (품질우선형) | 기술력, 안전, 품질인증 중시 |
| Relationship-driven (관계중시형) | 신뢰, 장기파트너십, 대응속도 |
| Innovation-seeking (혁신추구형) | 신기술, 자동화, 데이터 기반 관리 |

### 1.2 Effective Segmentation Criteria (효과적 세분화 5대 기준)

A segment must pass all five tests to be strategically useful:

```
┌────────────────────────────────────────────────────────┐
│              5 Criteria for Valid Segments              │
├──────────────────┬─────────────────────────────────────┤
│ Measurable       │ 크기, 구매력, 특성을 측정할 수 있는가?   │
│ (측정가능성)      │ Can you measure size & purchasing     │
│                  │ power of the segment?                 │
├──────────────────┼─────────────────────────────────────┤
│ Accessible       │ 효과적으로 접근하고 서비스할 수 있는가?   │
│ (접근가능성)      │ Can you reach and serve the segment?  │
├──────────────────┼─────────────────────────────────────┤
│ Substantial      │ 별도 마케팅 프로그램의 수익성이 있는가?   │
│ (충분성)          │ Is the segment large/profitable       │
│                  │ enough to serve?                      │
├──────────────────┼─────────────────────────────────────┤
│ Differentiable   │ 다른 세그먼트와 구별되게 반응하는가?     │
│ (차별성)          │ Does it respond differently to        │
│                  │ different marketing mixes?             │
├──────────────────┼─────────────────────────────────────┤
│ Actionable       │ 세그먼트를 공략하는 프로그램을 설계할     │
│ (실행가능성)      │ 수 있는가?                             │
│                  │ Can you design effective programs?     │
└──────────────────┴─────────────────────────────────────┘
```

## 2. Targeting (목표시장 선정)

Select which segment(s) to enter and how to serve them.

### 2.1 Five Targeting Strategies (5가지 타겟팅 전략)

```
Coverage Spectrum:
 Broad ◄──────────────────────────────────────────► Narrow

 ┌──────────┐ ┌──────────┐ ┌──────────┐ ┌────────┐ ┌────────┐
 │   Mass   │ │ Differen │ │  Concen  │ │ Micro  │ │One-to- │
 │ Marketing│ │  tiated  │ │ trated / │ │Marketin│ │  One   │
 │ 비차별적  │ │ 차별적    │ │  Niche   │ │ 마이크로 │ │ 일대일  │
 │          │ │          │ │ 집중/틈새 │ │        │ │        │
 └──────────┘ └──────────┘ └──────────┘ └────────┘ └────────┘
  전체시장     여러 세그먼트  하나의 세그먼트  지역/개인   개별고객
  단일 제안    각각 맞춤제안  집중 공략       맞춤       완전 맞춤
```

| Strategy | Description | When to Use | Risk |
|----------|-------------|-------------|------|
| **Mass (비차별적)** | 전체 시장에 동일 제안. 규모의 경제 활용 | 보편적 니즈, 자원 풍부 | 경쟁자의 틈새 공략에 취약 |
| **Differentiated (차별적)** | 여러 세그먼트에 각각 맞춤 마케팅 믹스 제공 | 자원 충분, 세그먼트 간 차이 큼 | 비용 증가, 자원 분산 |
| **Concentrated/Niche (집중/틈새)** | 1~2개 세그먼트에 자원 집중. 깊은 전문성 확보 | 자원 제한, 전문화 가능 | 시장 변동에 취약 |
| **Micro (마이크로)** | 지역/개인 그룹 단위 맞춤. 로컬 마케팅 | 고객 근접성 높음 | 규모의 경제 상실 |
| **One-to-One (일대일)** | 개별 고객 완전 맞춤. Mass customization | 데이터 풍부, 기술 지원 | 구현 복잡, 비용 최고 |

### 2.2 Segment Attractiveness Evaluation (세그먼트 매력도 평가)

Rate each segment on these criteria:

| Factor (평가요인) | Weight | Description |
|-------------------|--------|-------------|
| Market size (시장규모) | High | 현재 세그먼트의 총 규모 |
| Growth rate (성장률) | High | 향후 3~5년 성장 전망 |
| Profitability (수익성) | High | 마진, 가격 민감도, 원가구조 |
| Competition intensity (경쟁강도) | Medium | 경쟁사 수, 진입장벽 |
| Company fit (자사적합도) | High | 역량, 기술, 브랜드와의 적합성 |
| Risk (리스크) | Medium | 규제, 기술 변화, 의존도 |

### 2.3 Targeting Decision Matrix (타겟팅 의사결정 매트릭스)

```
                    Segment Attractiveness (세그먼트 매력도)

                ┌────────────┬────────────┬────────────┐
       High     │  Selective  │   Invest   │  PRIMARY   │

  Strength      ├────────────┼────────────┼────────────┤

                │  철수       │   수확      │  선별투자   │
                └────────────┴────────────┴────────────┘
```

## 3. Positioning (포지셔닝)

Define how your brand occupies a distinct place in the target customer's mind relative to competitors.

### 3.1 Perceptual Map (지각도/포지셔닝 맵)

A 2D map showing how customers perceive competing brands on key dimensions:

```
                         High Price (고가격)
                              ^
                              |
                    Brand C   |   Brand A
                      ★       |     ★
                              |
   Low Quality ───────────────┼──────────────── High Quality
   (저품질)                   |                  (고품질)
                              |
                    Brand D   |   Brand B
                      ★       |     ★
                              |
                              |    ◎ = Your ideal position
                              |        (목표 포지션)
                         Low Price (저가격)

   ★ = Competitor position (경쟁사 현재 위치)
   ◎ = Target position (목표 포지션)

   Finding "white space" (공백 지대) = strategic opportunity
```

### 3.2 Positioning Statement Template (포지셔닝 선언문)

```
┌─────────────────────────────────────────────────────────────┐
│                   Positioning Statement                      │
│                   포지셔닝 선언문 템플릿                       │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  For [target segment]          (목표 고객)                    │
│  who [need/want],              (니즈/원하는 것)               │
│  [Brand] is the [category]     (브랜드는 ~카테고리)           │
│  that [key benefit]            (핵심 편익을 제공하는)          │
│  because [reason to believe].  (그 이유는 ~이기 때문)         │
│                                                              │
├─────────────────────────────────────────────────────────────┤
│  Example:                                                    │
│                                                              │
│  For industrial plant managers                               │
│  who need reliable maintenance with zero safety incidents,   │
│  DY Industrial is the industrial cleaning specialist         │
│  that delivers data-driven, on-time project completion       │

│  project tracking technology.                                │
│                                                              │
│  (산업플랜트 관리자 중 무재해 안전과 신뢰할 수 있는 정비가       │
│   필요한 고객을 위해, DY산업개발은 데이터 기반으로 납기를        │
│   준수하는 산업청소 전문기업입니다. 20년 이상의 현장 노하우와    │

```

### 3.3 Points of Parity vs Points of Difference

```
┌───────────────────────────────────────────────────────┐
│       POP vs POD Analysis (동등점 vs 차별점)           │
├──────────────────────┬────────────────────────────────┤
│  POP (동등점)         │  POD (차별점)                   │
│  Points of Parity    │  Points of Difference          │
├──────────────────────┼────────────────────────────────┤
│ Associations NOT     │ Associations that are UNIQUE   │
│ unique to the brand  │ to the brand                   │
│ — shared with        │ — strongly held and favorably  │
│ competitors          │ evaluated by customers         │
├──────────────────────┼────────────────────────────────┤
│ "We're just as good  │ "We're better/different        │
│  as them at this"    │  because of this"              │
├──────────────────────┼────────────────────────────────┤
│ Category POP:        │ Must be:                       │
│ Minimum requirements │ • Desirable (고객에게 바람직)    │
│ to compete           │ • Deliverable (기업이 제공가능)  │
│                      │ • Differentiating (차별화됨)    │
│ Competitive POP:     │                                │
│ Negate competitor's  │                                │
│ claimed POD          │                                │
└──────────────────────┴────────────────────────────────┘
```

**POD Validity Test (차별점 유효성 검증):**

| Criterion | Question |
|-----------|----------|
| Relevance (적절성) | 고객이 실제로 중요하게 생각하는가? |
| Distinctiveness (독특성) | 경쟁사가 쉽게 모방할 수 없는가? |
| Believability (신뢰성) | 고객이 믿을 수 있는 근거가 있는가? |
| Feasibility (실현가능성) | 실제로 전달할 역량이 있는가? |
| Communicability (전달가능성) | 고객에게 쉽게 전달/이해 가능한가? |
| Sustainability (지속가능성) | 장기적으로 유지할 수 있는가? |

### 3.4 Repositioning Strategies (재포지셔닝 전략)

| Strategy | Description | When to Use |
|----------|-------------|-------------|
| **Image repositioning (이미지 재포지셔닝)** | 제품 동일, 이미지/커뮤니케이션 변경 | 인식과 실체의 괴리 |
| **Product repositioning (제품 재포지셔닝)** | 제품 변경, 동일 시장 유지 | 니즈 변화, 기술 발전 |
| **Intangible repositioning (무형 재포지셔닝)** | 동일 제품으로 새 세그먼트 공략 | 현 시장 성장 한계 |
| **Tangible repositioning (유형 재포지셔닝)** | 새 제품으로 새 세그먼트 진입 | 기존 포지션 한계, 다각화 |

## Output Format

When performing STP analysis, structure the output as follows:

### 🔬 Segmentation (시장세분화)
- Market definition and scope
- Segmentation bases used and rationale
- Segment profiles (3~6 segments)
- Segment size and growth estimates

### 🎯 Targeting (목표시장 선정)
- Segment attractiveness scoring
- Targeting decision matrix results
- Selected targeting strategy and rationale
- Priority segments (primary, secondary, monitor)

### 📍 Positioning (포지셔닝)
- Perceptual map with competitor positions
- Points of Parity (POP) and Points of Difference (POD)
- Positioning statement
- Key messages and proof points

### 📋 STP Summary & Recommendations
- Strategy overview (1-paragraph)
- Immediate actions (next 90 days)
- Marketing mix implications (4P alignment)
- Risks and contingencies

## Sub-commands

### `/think:stp` — Full STP Analysis
Runs the complete Segmentation → Targeting → Positioning pipeline.

**Input needed:**
- Industry/market context
- Company capabilities and current position
- Key competitors
- Available data (market research, customer data, financials)

### `/think:stp:segment` — Segmentation Analysis Only
Deep-dive into market segmentation. Produces detailed segment profiles without targeting or positioning decisions.

**Useful when:**
- Exploring a new market
- Existing segmentation feels outdated
- Need to discover unserved segments

### `/think:stp:position` — Positioning Strategy Only
Develops positioning strategy for an already-defined target segment.

**Input needed:**
- Target segment description
- Competitor landscape
- Company strengths and capabilities
- Customer decision criteria

**Produces:**
- Perceptual map
- POP/POD analysis
- Positioning statement
- Repositioning options (if current position exists)
