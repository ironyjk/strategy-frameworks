# Marketing Mix 4P/7P Framework

마케팅 믹스 — 마케팅 전략의 **전술적 실행 계층(tactical execution layer)**.
Jerome McCarthy(1960)의 4P + Booms & Bitner(1981)의 서비스 확장 7P.

## Architecture

```
┌─────────────────────────────────────────────────────────┐
│                   TARGET MARKET (표적시장)                │
│                                                         │
│   ┌───────────┐  ┌───────────┐                          │
│   │  Product   │  │   Price   │                          │
│   │  (제품)    │  │  (가격)    │     ◄── Core 4P         │
│   └─────┬─────┘  └─────┬─────┘         (McCarthy)       │
│         │              │                                │
│         ▼              ▼                                │
│   ┌─────────────────────────┐                           │
│   │     CUSTOMER VALUE      │                           │
│   │      (고객 가치)         │                           │
│   └─────────────────────────┘                           │
│         ▲              ▲                                │
│         │              │                                │
│   ┌───────────┐  ┌───────────┐                          │
│   │   Place   │  │ Promotion │                          │
│   │  (유통)    │  │  (촉진)    │     ◄── Core 4P         │
│   └───────────┘  └───────────┘                          │
│                                                         │
│   ─ ─ ─ ─ ─ ─ ─ Extended for Services ─ ─ ─ ─ ─ ─ ─   │
│                                                         │
│   ┌───────────┐  ┌───────────┐  ┌────────────────┐     │
│   │  People   │  │  Process  │  │   Physical     │     │
│   │  (사람)    │  │ (프로세스) │  │   Evidence     │     │
│   │           │  │           │  │  (물리적 증거)  │     │
│   └───────────┘  └───────────┘  └────────────────┘     │
│                                                         │
│              ◄── Extended 7P (Booms & Bitner) ──►       │
└─────────────────────────────────────────────────────────┘

        4C Alternative View (Lauterborn, 1990)
        ┌──────────────────────────────────┐
        │  Product  →  Customer Value      │
        │  Price    →  Cost to Customer    │
        │  Place    →  Convenience         │
        │  Promotion → Communication       │
        └──────────────────────────────────┘
```

## 1. 📦 Product (제품/서비스)

고객의 필요와 욕구를 충족시키는 유형/무형의 제공물.

### 제품 3계층 (Kotler's Three Levels)

```
┌─────────────────────────────────────┐
│        Augmented Product            │
│        (확장 제품)                   │
│   warranty, delivery, after-service │
│                                     │
│   ┌─────────────────────────────┐   │
│   │     Actual Product          │   │
│   │     (실제 제품)              │   │
│   │   brand, design, quality,   │   │
│   │   features, packaging       │   │
│   │                             │   │
│   │   ┌─────────────────────┐   │   │
│   │   │   Core Product      │   │   │
│   │   │   (핵심 제품)        │   │   │
│   │   │   core benefit      │   │   │
│   │   │   the customer is   │   │   │
│   │   │   really buying     │   │   │
│   │   └─────────────────────┘   │   │
│   └─────────────────────────────┘   │
└─────────────────────────────────────┘
```

- **핵심 제품 (Core)**: 고객이 진정으로 구매하는 편익. "드릴이 아니라 구멍을 산다."
- **실제 제품 (Actual)**: 브랜드, 디자인, 품질 수준, 기능, 포장
- **확장 제품 (Augmented)**: 보증, 배송, 설치, A/S, 신용

### 제품 수명주기 (Product Life Cycle)

```
  매출/이익
  ▲
  │         ★ 성장        ★ 성숙
  │        /  Growth      /  Maturity
  │       /              /─────────\
  │      /              /           \  ★ 쇠퇴
  │     /              /             \  Decline
  │    /              /               \
  │   / ★ 도입       /                 \
  │  /  Introduction /                   \
  │ /               /                     \
  └──────────────────────────────────────────► 시간
```

| 단계 | 전략 초점 | 가격 | 촉진 |
|------|----------|------|------|
| 도입 (Introduction) | 인지도 구축 | 스키밍 or 침투 | 높은 광고비 |
| 성장 (Growth) | 시장점유율 확대 | 경쟁 대응 | 브랜드 차별화 |
| 성숙 (Maturity) | 이익 극대화 | 경쟁 가격 | 리마인더 광고 |
| 쇠퇴 (Decline) | 비용 절감 or 철수 | 할인 | 최소화 |

### 제품 믹스 의사결정

- **제품 폭 (Width/Breadth)**: 제품 라인 수
- **제품 깊이 (Depth)**: 각 라인 내 품목 수
- **제품 길이 (Length)**: 전체 품목 수 합계
- **일관성 (Consistency)**: 제품 라인 간 관련성

## 2. 💰 Price (가격)

고객이 제품/서비스를 획득하기 위해 지불하는 총 비용.

### 가격 결정 전략

| 전략 | 설명 | 적용 상황 |
|------|------|----------|
| **원가 가산 (Cost-Plus)** | 원가 + 마크업 % | 제조업, B2B 입찰 |
| **가치 기반 (Value-Based)** | 고객 인지 가치 기준 | 차별화 제품, 프리미엄 |
| **경쟁 기반 (Competitive)** | 경쟁사 가격 참조 | 성숙 시장, 동질 제품 |
| **침투 가격 (Penetration)** | 낮은 초기 가격으로 시장 진입 | 신규 시장, 규모의 경제 |
| **스키밍 (Skimming)** | 높은 초기 가격, 점진 인하 | 혁신 제품, 초기 수용자 |
| **심리적 가격 (Psychological)** | 9,900원 vs 10,000원 | B2C, 소비재 |
| **동적 가격 (Dynamic)** | 수요/시간대별 변동 | 항공, 호텔, 이커머스 |
| **프리미엄 (Premium)** | 의도적 고가 포지셔닝 | 럭셔리, 전문 서비스 |

### 가격 탄력성 (Price Elasticity of Demand)

```
  Ed = (% 수요량 변화) / (% 가격 변화)

  |Ed| > 1  →  탄력적 (Elastic): 가격 인하 시 매출 증가
  |Ed| < 1  →  비탄력적 (Inelastic): 가격 인상 시 매출 증가
  |Ed| = 1  →  단위 탄력적 (Unit Elastic)
```

**비탄력적 제품 특성**: 필수품, 대체재 없음, 브랜드 충성도 높음, 지출 비중 낮음
**탄력적 제품 특성**: 사치품, 대체재 많음, 전환 비용 낮음, 지출 비중 높음

## 3. 🏪 Place (유통/장소)

제품이 고객에게 도달하는 경로와 방식.

### 유통 채널 구조

```
  직접 유통 (Direct)
  ┌──────────┐                    ┌──────────┐
  │ 제조업체  │ ──────────────────► │  소비자   │
  │ Producer │                    │ Consumer │
  └──────────┘                    └──────────┘

  간접 유통 1단계
  ┌──────────┐    ┌──────────┐    ┌──────────┐
  │ 제조업체  │ ──►│  소매상   │ ──►│  소비자   │
  │ Producer │    │ Retailer │    │ Consumer │
  └──────────┘    └──────────┘    └──────────┘

  간접 유통 2단계
  ┌──────────┐  ┌──────────┐  ┌──────────┐  ┌──────────┐
  │ 제조업체  │►│  도매상   │►│  소매상   │►│  소비자   │
  │ Producer │  │Wholesaler│  │ Retailer │  │ Consumer │
  └──────────┘  └──────────┘  └──────────┘  └──────────┘
```

### 채널 전략

| 전략 | 설명 | 예시 |
|------|------|------|
| **집중적 유통 (Intensive)** | 최대한 많은 판매점 | 음료, 스낵, 생필품 |
| **선택적 유통 (Selective)** | 선별된 유통 파트너 | 가전, 의류, 화장품 |
| **전속적 유통 (Exclusive)** | 제한된 소수 판매점 | 럭셔리, 자동차, 프랜차이즈 |

### 직접 vs 간접 비교

| 구분 | 직접 (Direct) | 간접 (Indirect) |
|------|--------------|-----------------|
| 통제력 | 높음 | 낮음 |
| 마진 | 높음 | 중간 경유 마진 |
| 도달 범위 | 제한적 | 넓음 |
| 초기 투자 | 높음 | 낮음 |
| 고객 데이터 | 직접 확보 | 간접적 |
| 예시 | D2C, 자사 쇼핑몰, 영업사원 | 유통사, 대리점, 마켓플레이스 |

## 4. 📣 Promotion (촉진)

고객에게 제품/서비스를 알리고 구매를 유도하는 모든 커뮤니케이션 활동.

### 촉진 믹스 (Promotional Mix)

| 수단 | 설명 | 장점 | 단점 |
|------|------|------|------|
| **광고 (Advertising)** | 유료 매체를 통한 비인적 커뮤니케이션 | 대규모 도달 | 고비용, 일방향 |
| **PR (Public Relations)** | 언론, 이벤트, 사회공헌 | 신뢰도 높음 | 통제 어려움 |
| **판매촉진 (Sales Promotion)** | 쿠폰, 할인, 샘플, 경품 | 즉각 반응 | 단기적, 브랜드 훼손 위험 |
| **인적 판매 (Personal Selling)** | 영업사원 직접 대면 | 맞춤 대응, 관계 | 고비용, 도달 제한 |
| **디지털 마케팅 (Digital)** | SEO, SNS, 콘텐츠, 이메일 | 측정 가능, 타겟팅 | 경쟁 과열, 알고리즘 변화 |
| **다이렉트 마케팅 (Direct)** | DM, 텔레마케팅, 카탈로그 | 개인화 | 스팸 인식 위험 |

### 통합 마케팅 커뮤니케이션 (IMC)

**Integrated Marketing Communications** — 모든 촉진 수단을 일관된 메시지로 통합.

```
                    ┌─────────────────┐
                    │  Brand Message  │
                    │  (브랜드 메시지) │
                    └────────┬────────┘
                             │
           ┌─────────────────┼─────────────────┐
           │                 │                 │
     ┌─────▼─────┐   ┌──────▼──────┐   ┌──────▼──────┐
     │  온라인    │   │  오프라인    │   │  경험/체험   │
     │  Online   │   │  Offline    │   │ Experience  │
     ├───────────┤   ├─────────────┤   ├─────────────┤
     │ SNS       │   │ TV/라디오   │   │ 이벤트      │
     │ 검색광고   │   │ 인쇄매체    │   │ 팝업스토어   │
     │ 콘텐츠    │   │ 옥외광고    │   │ 전시회      │
     │ 이메일    │   │ DM         │   │ 체험 마케팅  │
     └───────────┘   └─────────────┘   └─────────────┘
                             │
                    ┌────────▼────────┐
                    │  Consistent     │
                    │  ONE VOICE      │
                    │  (일관된 메시지)  │
                    └─────────────────┘
```

**IMC 핵심 원칙**:
1. 고객 접점(touchpoint) 전체에 걸친 메시지 일관성
2. 채널 간 시너지 극대화
3. 고객 여정(customer journey) 기반 설계
4. 측정 가능한 ROI 추적

## 5. 🤝 People (사람) — 서비스 7P 확장

서비스 제공에 관여하는 모든 사람 (직원, 고객, 기타 고객).

### 핵심 요소

- **직원 교육 (Employee Training)**: 서비스 품질의 핵심 결정 요인
- **내부 마케팅 (Internal Marketing)**: 직원을 첫 번째 고객으로 대우
- **고객 상호작용 (Customer Interaction)**: 서비스 접점(MOT: Moment of Truth) 관리
- **서비스 문화 (Service Culture)**: 고객 중심 조직 문화 구축
- **채용 기준 (Hiring Criteria)**: 기술 + 태도 + 서비스 마인드

### 서비스 이익 사슬 (Service-Profit Chain)

```
내부 서비스 품질 → 직원 만족 → 직원 유지 → 외부 서비스 품질
                                              → 고객 만족 → 고객 충성 → 수익/성장
```

## 6. ⚙️ Process (프로세스) — 서비스 7P 확장

서비스가 전달되는 절차, 메커니즘, 활동의 흐름.

### 핵심 요소

- **서비스 청사진 (Service Blueprinting)**: 고객 행동 + 프론트/백 스테이지 + 지원 프로세스
- **표준화 vs 맞춤화 (Standardization vs Customization)**: 일관성과 유연성의 균형
- **대기 관리 (Queue Management)**: 대기 시간 인식 관리, 예약 시스템
- **자동화 (Automation)**: 셀프서비스, 키오스크, 챗봇, 온라인 예약
- **실패 복구 (Service Recovery)**: 서비스 실패 시 회복 프로토콜

### 서비스 청사진 구조

```
┌─────────────────────────────────────────────────────┐
│  고객 행동 (Customer Actions)                        │
│  도착 → 대기 → 주문 → 수령 → 사용 → 피드백           │
├─────── Line of Interaction ─────────────────────────┤
│  전방 무대 (Onstage / Front Stage)                   │
│  접수 → 안내 → 상담 → 서비스 제공 → 후속 연락        │
├─────── Line of Visibility ──────────────────────────┤
│  후방 무대 (Backstage)                               │
│  준비 → 품질 확인 → 재고 관리 → 보고서 작성           │
├─────── Line of Internal Interaction ────────────────┤
│  지원 프로세스 (Support Processes)                    │
│  IT 시스템 → CRM → 물류 → 교육 → 인사 관리           │
└─────────────────────────────────────────────────────┘
```

## 7. 🏢 Physical Evidence (물리적 증거) — 서비스 7P 확장

서비스의 무형성을 보완하는 유형적 단서와 환경.

### 핵심 요소

- **서비스스케이프 (Servicescape)**: 서비스가 전달되는 물리적 환경 (Bitner, 1992)
  - 주변 조건 (ambient): 온도, 조명, 음악, 향기
  - 공간 배치 (spatial layout): 동선, 장비 배치, 기능성
  - 표지/상징 (signs & symbols): 간판, 인테리어, 유니폼
- **유형적 단서 (Tangible Cues)**:
  - 브로슈어, 명함, 보고서 품질
  - 웹사이트, 앱 인터페이스
  - 유니폼, 장비 상태
  - 시설 청결도, 인테리어
- **디지털 물리적 증거**: UI/UX, 리뷰, 인증서, 포트폴리오

## 8. 🔄 4C 대안적 관점 (Lauterborn, 1990)

기업 관점(4P)에서 **고객 관점(4C)**으로의 전환.

| 4P (기업 관점) | 4C (고객 관점) | 전환 의미 |
|:---:|:---:|---|
| Product (제품) | **Customer Value (고객 가치)** | "무엇을 만들까" → "고객이 무엇을 원하는가" |
| Price (가격) | **Cost to Customer (고객 비용)** | 가격표 → 총 소유 비용 (시간, 노력, 심리적 비용 포함) |
| Place (유통) | **Convenience (편의성)** | 유통 채널 → 고객이 가장 편하게 구매하는 방법 |
| Promotion (촉진) | **Communication (소통)** | 일방적 광고 → 쌍방향 대화와 관계 구축 |

**적용 팁**: 4P로 기획하되, 4C 렌즈로 검증하라.

## Sub-Commands

### `/think:marketing-mix` — Full 7P Analysis
전체 7P 마케팅 믹스 분석. 서비스업 포함 모든 산업에 적용.

**절차**:
1. 대상 기업/제품/서비스 파악
2. 4P 각각에 대한 현황 분석
3. 서비스 요소 존재 시 7P 확장 분석
4. 4C 관점 교차 검증
5. 마케팅 믹스 일관성(coherence) 평가
6. 핵심 권고사항 도출

### `/think:marketing-mix:4p` — 4P Product-Focused Analysis
제품(유형재) 중심의 4P 분석. 제조업, 소비재, B2B 제품에 최적화.

**초점**:
- 제품 3계층 심층 분석
- PLC 단계 판단 및 전략 제안
- 제품 믹스(폭/깊이/길이) 평가
- 가격-품질 매트릭스 매핑
- 유통 채널 효율성

### `/think:marketing-mix:service` — Service-Specific 7P Analysis
서비스업 특화 7P 분석. IHIP 특성(무형성, 이질성, 비분리성, 소멸성) 중심.

**초점**:
- 서비스 청사진 작성
- MOT(Moment of Truth) 식별 및 관리
- 서비스 품질 갭 모델 (SERVQUAL)
- People/Process/Physical Evidence 심층 분석
- 서비스 이익 사슬 평가

### `/think:marketing-mix:audit` — Marketing Mix Audit
현재 마케팅 믹스의 정합성(coherence)과 효과성을 감사.

**감사 항목**:
1. **내적 일관성**: 4P/7P 요소 간 정합성 (가격↔포지셔닝, 유통↔타겟 등)
2. **외적 적합성**: 시장 환경, 경쟁 상황, 고객 니즈와의 적합도
3. **전략 정렬**: STP(세분화-타겟팅-포지셔닝)와의 정렬
4. **갭 분석**: 현재 상태 vs 이상적 상태
5. **우선순위 매트릭스**: 영향도 x 실행 용이성 기반 개선 우선순위

**출력**: 4P/7P 각 요소별 점수(1~5), 일관성 매트릭스, 개선 로드맵
