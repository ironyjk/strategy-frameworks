# Resource-Based View (자원기반관점)

> "Firms are bundles of productive resources... the heterogeneity of resources across firms explains performance differences."
> — Edith Penrose (1959), Birger Wernerfelt (1984), Jay Barney (1991)

## 1. Core Premise (핵심 전제)

Firms are **bundles of resources and capabilities**. Performance differences arise because:
- Resources are **heterogeneously** distributed across firms (자원 이질성)
- Resources are **imperfectly mobile** — they cannot be freely traded (불완전 이동성)
- Competitive advantage comes from **inside-out**, not just outside-in

This contrasts with Porter's industry-structure view: RBV says **what you have** matters more than **where you compete**.

**Key insight**: Tangible resources are easiest to copy. **Intangible + Human + Organizational** resources are where sustainable advantage lives.

## 3. VRIO Framework (VRIO 프레임워크)

The VRIO test determines whether a resource can generate **sustained competitive advantage** (지속적 경쟁우위).

### 3.1 Four Questions

| Test | Question | 핵심 질문 |
|------|----------|-----------|
| **V** — Valuable (가치 있는) | Does it exploit an opportunity or neutralize a threat? | 이 자원이 기회를 활용하거나 위협을 무력화하는가? |
| **R** — Rare (희소한) | Do few competitors possess it? | 소수의 경쟁자만 보유하고 있는가? |
| **I** — Inimitable (모방 불가한) | Is it costly to imitate? | 모방하는 데 비용이 많이 드는가? |
| **O** — Organized (조직화된) | Is the firm organized to capture value? | 이 자원의 가치를 포착하도록 조직되어 있는가? |

### 3.2 Three Reasons for Inimitability (모방 불가의 3가지 원인)

1. **Unique Historical Conditions (독특한 역사적 조건)**: Path-dependent accumulation — "you had to be there"
2. **Causal Ambiguity (인과적 모호성)**: Competitors cannot identify which resources cause advantage
3. **Social Complexity (사회적 복잡성)**: Culture, trust, relationships cannot be engineered

### 3.3 VRIO Decision Matrix

```
 V? │ R? │ I? │ O? │ Competitive Implication        │ Performance
────┼────┼────┼────┼────────────────────────────────┼──────────────
 ✗  │  — │  — │  — │ ❌ Disadvantage (경쟁열위)      │ Below Normal
 ✓  │ ✗  │  — │  — │ ➖ Parity (경쟁균형)            │ Normal
 ✓  │ ✓  │ ✗  │  — │ ⏳ Temporary Advantage (일시적) │ Above Normal
 ✓  │ ✓  │ ✓  │ ✗  │ ⚠️  Unexploited (미활용)        │ Below Potential
 ✓  │ ✓  │ ✓  │ ✓  │ 🏆 Sustained Advantage (지속적) │ Above Normal
```

## 4. Dynamic Capabilities (동적 역량) — Teece (1997, 2007)

When environments change rapidly, **static resources decay**. Dynamic capabilities extend RBV:

| Capability | Description | 설명 |
|------------|-------------|------|
| **Sensing (감지)** | Scan, learn, interpret — identify opportunities and threats | 환경 변화를 감지하고 기회를 탐색 |
| **Seizing (포착)** | Design business models, commit resources, mobilize | 기회를 포착하여 자원을 투입 |
| **Transforming (전환)** | Reconfigure assets, realign structure, renew routines | 자산을 재배치하고 조직을 변환 |

**When RBV alone isn't enough**: In fast-moving industries (tech, platform), static resources lose value quickly. Dynamic capabilities = the ability to **build, integrate, and reconfigure** resources over time.

## 5. Core Competence Integration (핵심역량 — Prahalad & Hamel, 1990)

Core competence is the **collective learning** of the organization. Three tests:

| Test | Question | 핵심 질문 |
|------|----------|-----------|
| **Customer Benefit (고객 가치)** | Does it significantly contribute to perceived customer benefits? | 고객이 인식하는 가치에 크게 기여하는가? |
| **Competitor Differentiation (경쟁 차별화)** | Is it difficult for competitors to imitate? | 경쟁자가 모방하기 어려운가? |
| **Extendability (확장 가능성)** | Can it be leveraged across markets and products? | 여러 시장과 제품으로 확장 가능한가? |

**Relationship to VRIO**: Core competence = a resource that passes all four VRIO tests AND can be applied across multiple business lines.

## 6. Resource Audit Process (자원 감사 프로세스)

### Step-by-step

```
Step 1: IDENTIFY (식별)
  └─ List all resources the firm possesses or controls

Step 2: CLASSIFY (분류)
  └─ Categorize as Tangible / Intangible / Human / Organizational

Step 3: VRIO TEST (VRIO 테스트)
  └─ Apply V → R → I → O sequentially to each resource

Step 4: STRATEGIC IMPLICATIONS (전략적 시사점)
  ├─ Protect resources with sustained advantage (보호)
  ├─ Invest in resources with temporary advantage → build inimitability (투자)
  ├─ Monitor parity resources → don't overspend (유지)
  └─ Divest or ignore disadvantage resources (정리)
```

## 7. RBV vs Porter: Inside-Out vs Outside-In (비교)

```
┌──────────────────┬────────────────────────┬────────────────────────┐
│ Dimension        │ RBV (Inside-Out)       │ Porter (Outside-In)    │
│                  │ 자원기반관점           │ 산업구조관점           │
├──────────────────┼────────────────────────┼────────────────────────┤
│ Starting point   │ Internal resources     │ Industry structure     │
│ Key question     │ "What do we have?"     │ "Where should we be?"  │
│ Unit of analysis │ Firm                   │ Industry               │
│ Advantage source │ Resource heterogeneity │ Market positioning     │
│ Time horizon     │ Long-term accumulation │ Strategic repositioning│
│ Best for         │ Diversification,       │ Entry decisions,       │
│                  │ capability building    │ competitive moves      │
│ Weakness         │ Tautology risk,        │ Ignores internal       │
│                  │ hard to measure        │ resource differences   │
└──────────────────┴────────────────────────┴────────────────────────┘
```

### When to Use Which

- **Use RBV** when: deciding what to build/acquire, evaluating diversification, assessing "can we compete?"
- **Use Porter** when: choosing which market to enter, analyzing competitive dynamics, pricing strategy
- **Integrate both** for complete strategy: Porter tells you WHERE to play, RBV tells you HOW to win

## 8. Output Format

Analysis output uses the following structure with emoji headers:

```
🔍 자원 감사 (Resource Audit)
  - Identified resources by category

📊 VRIO 분석 (VRIO Analysis)
  - VRIO table for each resource
  - Competitive implication per resource

🏆 핵심역량 (Core Competencies)
  - Resources passing all VRIO tests
  - Core competence 3-test evaluation

⚡ 동적 역량 (Dynamic Capabilities)
  - Sensing / Seizing / Transforming assessment
  - Capability gaps and development priorities

🎯 전략적 시사점 (Strategic Implications)
  - Protect, invest, maintain, divest recommendations
  - Resource development roadmap

🔗 외부 연계 (External Integration)
  - Link to Porter's Five Forces / Value Chain
  - Industry context for resource value
```

## 9. Sub-Commands

### `/rbv` — Full Resource Analysis (전체 자원 분석)
Complete resource-based analysis: identify → classify → VRIO → core competence → dynamic capabilities → strategic implications.

### `/rbv:vrio` — VRIO Assessment (VRIO 평가)
Focused VRIO analysis of specific resources. Input: list of resources or capabilities. Output: VRIO table with competitive implications.

### `/rbv:capability` — Dynamic Capabilities Audit (동적 역량 감사)
Assess the firm's sensing, seizing, and transforming capabilities. Identify capability gaps and prioritize development. Evaluate readiness for environmental change.
